    Name: Text Encryption Program

    Description:
    • Encrypts and decrypts files for user using Vigenere or Playfair cyphers
    • Outputs encrypted text to desired directory for user
    • Vigenere cypher shifts letters in accordance to value of respective index in keyword
    • Playfair swaps pairs of letters in accordance to cypher tableau which is either randomly generated and saved or created sequentailly after insertion of user keyword
    Development:
    • Java
    • Netbeans IDE
    • Sept 2016

    View Source Code: Click Here
